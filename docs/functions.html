üî§ String Functions

UPPER(string)
Converts a string to uppercase
UPPER(expression)

Parameters:

  • expression - String expression to convert

Example:

SELECT UPPER(first_name) FROM employees
"john" ‚Üí "JOHN"
LOWER(string)
Converts a string to lowercase
LOWER(expression)

Example:

SELECT LOWER(last_name) FROM employees
"SMITH" ‚Üí "smith"
LENGTH(string) / LEN(string)
Returns the length of a string
LENGTH(expression)

Example:

SELECT LENGTH(email) FROM users
"user@example.com" ‚Üí 16
SUBSTR(string, start, length) / SUBSTRING(string, start, length)
Extracts a substring from a string (1-based indexing)
SUBSTR(expression, start_position, [length])

Parameters:

  • expression - Source string
  • start_position - Starting position (1-based)
  • length - Optional length to extract

Examples:

SUBSTR("Hello World", 1, 5) ‚Üí "Hello"
SUBSTR("Hello World", 7) ‚Üí "World"
CONCAT(string1, string2, ...)
Concatenates multiple strings together
CONCAT(expression1, expression2, [expression3, ...])

Example:

SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM employees
CONCAT("John", " ", "Doe") ‚Üí "John Doe"
TRIM(string) / LTRIM(string) / RTRIM(string)
Removes whitespace from strings
TRIM(expression) -- Remove leading and trailing spaces
LTRIM(expression) -- Remove leading spaces
RTRIM(expression) -- Remove trailing spaces

Examples:

TRIM(" hello ") ‚Üí "hello"
LTRIM(" hello ") ‚Üí "hello "
RTRIM(" hello ") ‚Üí " hello"
LEFT(string, length) / RIGHT(string, length)
Extract characters from left or right side of string
LEFT(expression, length)
RIGHT(expression, length)

Examples:

LEFT("Hello World", 5) ‚Üí "Hello"
RIGHT("Hello World", 5) ‚Üí "World"
REPLACE(string, old_string, new_string)
Replace all occurrences of a substring
REPLACE(expression, old_value, new_value)

Example:

REPLACE("Hello World", "World", "Universe") ‚Üí "Hello Universe"
REVERSE(string)
Reverses the characters in a string
REVERSE(expression)

Example:

REVERSE("Hello") ‚Üí "olleH"
REPEAT(string, count)
Repeats a string a specified number of times
REPEAT(expression, count)

Example:

REPEAT("Ha", 3) ‚Üí "HaHaHa"

üî¢ Math Functions

ABS(number)
Returns the absolute value of a number
ABS(expression)

Examples:

ABS(-5) ‚Üí 5

ROUND(number, precision)

Examples:

FLOOR(expression) -- Round down
CEIL(expression) -- Round up
CEILING(expression) -- Round up (alias)

Examples:

FLOOR(3.7) ‚Üí 3
CEIL(3.2) ‚Üí 4
MOD(dividend, divisor) / MODULO(dividend, divisor)
Returns the remainder of division
MOD(expression1, expression2)

Example:

MOD(10, 3) ‚Üí 1
POWER(base, exponent) / POW(base, exponent)

POWER(base, exponent)

Example:

POWER(2, 3) ‚Üí 8
SQRT(number)
Returns the square root of a number
SQRT(expression)

Example:

SQRT(16) ‚Üí 4
SIGN(number)
Returns the sign of a number (-1, 0, or 1)
SIGN(expression)

Examples:

SIGN(-5) ‚Üí -1
SIGN(0) ‚Üí 0
SIGN(3) ‚Üí 1

üîç Pattern Matching

LIKE Pattern Matching
Match strings using wildcards
column_name LIKE 'pattern'

Wildcards:

  • % - Matches any sequence of characters (including zero)
  • _ - Matches exactly one character
Starts With
name LIKE 'J%'
Matches: "John", "Jane", "Jack"
Ends With
name LIKE '%son'
Matches: "Johnson", "Anderson"
Contains
name LIKE '%an%'
Matches: "Jane", "Diana", "Frank"
Single Character
name LIKE 'J_n%'
Matches: "Jane", "June" but not "John"
Complex Pattern
email LIKE '%@gmail.%'
Matches Gmail addresses
Multiple Wildcards
name LIKE '_a%e'
Matches: "Jane", "Kate", "Dave"
NULL Testing
Test for null or empty values
column_name IS NULL
column_name IS NOT NULL

Examples:

SELECT * FROM employees WHERE manager_id IS NULL
SELECT * FROM users WHERE email IS NOT NULL
